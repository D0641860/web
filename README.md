# web
初體驗web製作，此專題為捐助網頁，分為前後部分，前端分為home.html主頁用於顯示此網站的資訊及各分頁的連結導向
donate.html為捐助網頁，可在裡面選填要捐助的物資及個人資料，傳至後端後，因此時還沒使用資料庫，虛手動更新物資目前狀態
list.html為清單網頁，此網頁列出目前的物資數量，讓有需要的人可以透過申請，經由我們後端人工審核後將物資寄送給有需要的人並更新物資清
comment.html為評論版網頁，瀏覽網頁的人可以在此網頁上留下自己想講的話
aboutus.html為關於我們的網頁，介紹做此網頁的動機
app.js為後端部分，我們利用node.js做後端，把donate.html及list.html的資料儲存至terminal並將網頁導回首頁

此專題為第一個多人一起協作的專題，深度雖然沒有很深，但卻理解了多人合作是如何運作的，及熟悉git的應用
